# Wistant

## كبير المسؤولين التقنيين & مهندس برمجيات

*التميز الهندسي على نطاق واسع • الأمن أولاً • تجربة المطور (DX)*

---

### ✦ الفلسفة

بناء أنظمة ليست فقط وظيفية، بل **مرنة**، **قابلة للتوسع**، و **بسيطة**. أؤمن بقوة تجربة المطور (DX) في دفع الهندسة عالية السرعة وضرورة الوضوح المعماري في المجالات المعقدة.

### ✦ المجموعة التقنية الاستراتيجية

| **الأسس المعمارية** | **هندسة الأنظمة** | **البيانات والبنية التحتية** |
| :--- | :--- | :--- |
| TypeScript / Node.js | هندسة الخدمات المصغرة | PostgreSQL / Redis |
| Next.js / React | الأنظمة الموزعة | Docker / Kubernetes |
| NestJS / GraphQL | تصميم واجهة برمجة التطبيقات (REST/gRPC) | Linux / Cloud Native |
